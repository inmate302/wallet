## Ya. Pero en serio, ¿Qué es Alke-Wallet? ##

Alke wallet es una billetera digital funcional. El proyecto encapsula el uso de la manipulación del DOM , Hojas de estilo y Javascript para poder presentar un producto funcional y responsivo.


## Guía Rápida ##

Descarga el archivo zip desde el botón "code" y extraelo en tu computador.
O si prefieres, usando la línea de comandos, clona el repo a un directorio usando el siguiente comando:

```
git clone https://github.com/inmate302/wallet.git
```

Una vez en la página de inicio deberías hacer click en el botón "iniciar sesión".
Después de eso deberías poder iniciar sesión usando las siguientes credenciales:

```
correo electrónico: lucho1985@gmail.com
contraseña: 1234

```
A continuación, te encontrarás con el menu principal:

    * Presiona "Depositar" para tus operaciones de depósito.
    * Presiona "Enviar dinero" para tus operaciones de transferencia de dinero.
    * Presiona "Últimos movimientos" para ver tu historial de transacciones.

**_Para el funcionamiento correcto de las funciones mencionadas, tienes que permitir las cookies en tu navegador. Una vez la base de datos sea implementada en el futuro, estas no serán necesarias._**
